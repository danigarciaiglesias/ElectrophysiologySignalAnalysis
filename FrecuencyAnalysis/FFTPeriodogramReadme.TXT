This is a Periodogram function for R.

It calculates signal's periodogram and the array of frecuencies of a given signal x. Also if plot is marked you can plot the periodogram of the given signall

Parameters used in this function are:
x :  signal given.
fs: sampling rate.
win: windowing array for the signal.
range: "onesided" or "twosided".
plot: 1 for plotting and 0 for no plotting.
